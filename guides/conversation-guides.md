# 대화가이드 (Conversation Guides)

일상 주제별 자연스러운 영어 대화 표현 모음

---

## 대화가이드 #01 — 점심 같이 가요

**상황:** 동료에게 점심을 같이 먹자고 제안하고, 잠깐 기다려 달라고 하는 상황

| 한국어 | 영어 |
|--------|------|
| 점심 먹으러 나갈까 하는데, 같이 가실래요? | I'm thinking of going out for some lunch right now, do you want to join me? |
| 뭐 좀 하고 있어서요, 조금만 기다려 줄래요? | I'm in the middle of something right now, could you wait for me a little bit? |
| 저 아래층에 가 있을게요 | OK, I'll be downstairs. |
| 최대한 빨리 갈게요 | I'll be there as soon as I can. |
| 괜찮아요, 천천히 하세요 | That's OK, take your time. |

**핵심 표현**
* **be thinking of + ing** — ~할까 하는데 (고려 중)
* **be in the middle of something** — ~하고 있는 중이다
* **go out for ~** — ~하러 나가다
* **as soon as I can** — 최대한 빨리

## 대화가이드 #02 — 주말 / 새 핸드폰

| 한국어 | 영어 |
|--------|------|
| 주말에 뭐했어? | What did you do last weekend? |
| 별거 안했어 | I didn't do much. |
| 핸드폰 바꾼다고 하지 않았어? | Didn't you say you were going to get a new phone? |
| 아 맞다! 짜잔~ | Oh, that's right! Check it out~ |
| 무슨 이유에서인지 계속 문제가 생겨 | But for some reason, it kept causing problems |
| 새거를 살 때가 됐다 싶었지 | So I decided it was time to get a new one |

**핵심 표현**
* **didn't you say ~?** — ~라고 하지 않았어?
* **check it out** — 짜잔~ / 이것봐라~
* **get + something + p.p.** — 사역 (맡기다/받다): _I got this phone fixed._
* **keep + ing** — 계속/자꾸 반복해서 일어나다

## 대화가이드 #03 — 전화 받을 때

| 한국어 | 영어 |
|--------|------|
| 안 받을 거야? 어서 받아 | Aren't you going to get that? Go ahead, get it. |
| 잠시만요, 통화 좀 하고 올게 | Could you hold on for a second? I need to take this call. |
| 삼촌이랑 친해? | Are you close with your uncle? |
| 가끔 전화를 하시더라고 | He's been calling me from time to time. |

**핵심 표현**
* **go ahead** — 어서 자유롭게 해
* **hold on** — 기다려줄래?
* **I need to take this call** — 전화 받아야 할 때
* **be close with someone** — ~와 친하다
* **from time to time** — 가끔

## 대화가이드 #04 — 처음 보는 강한 인상 표현

| 한국어 | 영어 |
|--------|------|
| 나 저렇게 고집 센 사람 처음 봐 | I've never seen such a stubborn person. |
| 나 이렇게 맛있는 파스타는 처음이야 | This is the best pasta I've ever had. |
| 저렇게 많이 먹는 사람 처음 봐 | I've never seen anyone eat so much. |

**강조 표현**
```
This is the best ~         (기본)
This has to be the best ~  (확신)
This must be the best ~    (강한 확신)
This has got to be the best ~ (가장 강한 강조)
```

## 대화가이드 #09 — 음식 사양하기

| 한국어 | 영어 |
|--------|------|
| 아뇨, 전 됐어요 | No, I'm good. |
| 왜 아무 말도 안 했어요? | Why didn't you say anything? |
| 다른 거 시켰을 텐데 | We would have gotten something else. |
| 다른 거 잘 먹고 있어요 | I'm really enjoying everything else. |

**핵심 표현**
* **I'm good** — 사양할 때 가장 자연스러운 표현
* **would have gotten** — ~했을 텐데 (가정법)
* **tons of = a lot of**
* **I'd really prefer it if ~** — ~했으면 정말 좋겠어

## 대화가이드 #10 — 예약 변경 요청

| 한국어 | 영어 |
|--------|------|
| 이번 주 내로 만날 수 있는 방법은 없을까요? | Is there any way I can see him this week? |
| 월요일이 제일 빨라요 | Monday is the best we can do. |
| 무리한 부탁인 건 아는데 | I know it's a lot to ask |
| 제가 할 수 있는 건 해보겠는데요 | I'll see what I can do |

**핵심 표현**
* **Is there any way I can ~?** — ~할 수 있는 방법이 있을까요?
* **a lot to ask** — 무리한 부탁
* **I'll see what I can do** — 할 수 있는 건 해볼게요
* **I can't make any promises**

## 대화가이드 #11 — 공포영화를 왜 좋아해?

**핵심 표현**
* **I haven't thought of it like that** — 그렇게는 생각 안 해봤는데
* **that makes sense** — 말이 되네, 이해돼

---

## 대화가이드 #16 — 쿠폰 / 할인 안내

**핵심 표현**
* **I hope that makes sense** — 이해 확인 (Do you understand?보다 자연스러움)
* **all I need to do is ~** — ~하기만 하면 된다
* **if I understood you correctly** — 제가 맞게 이해한 거라면

---

## 대화가이드 #17 — 낯이 익은 사람

| 한국어 | 영어 |
|--------|------|
| 저 사람 낯이 좀 익어서 | He looks kind of familiar. |
| 가서 말 걸어봐 | Why don't you go talk to him? |
| 아니야, 됐어 | Nah, it's OK. / Forget it. |

**핵심 표현**
* **look kind of familiar** — 낯이 익다
