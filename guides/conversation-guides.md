# 대화가이드 (Conversation Guides)

일상 주제별 자연스러운 영어 대화 표현 모음

---

## 대화가이드 #01 — 점심 같이 가요

**상황:** 동료에게 점심을 같이 먹자고 제안하고, 잠깐 기다려 달라고 하는 상황

| 한국어 | 영어 |
| --- | --- |
| 점심 먹으러 나갈까 하는데, 같이 가실래요? | I'm thinking of going out for some lunch right now, do you want to join me? |
| 뭐 좀 하고 있어서요, 조금만 기다려 줄래요? | I'm in the middle of something right now, could you wait for me a little bit? |
| 저 아래층에 가 있을게요 | OK, I'll be downstairs. |
| 최대한 빨리 갈게요 | I'll be there as soon as I can. |
| 괜찮아요, 천천히 하세요 | That's OK, take your time. |

**핵심 표현**

* **be thinking of + ing** — ~할까 하는데 (고려 중)

    * _I'm thinking of going out for a drive / a walk / dinner_

* **be in the middle of something** — ~하고 있는 중이다
* **go out for ~** — ~하러 나가다 _(go out for lunch / a drive / dinner)_
* **as soon as I can** — 최대한 빨리

---

## 대화가이드 #02 — 주말 / 새 핸드폰

**상황:** 주말에 뭐 했는지 묻고, 새 핸드폰을 보여주는 대화

| 한국어 | 영어 |
| --- | --- |
| 주말에 뭐했어? | What did you do last weekend? / How was your weekend? |
| 별거 안했어 | I didn't do much. / Not much. |
| 핸드폰 바꾼다고 하지 않았어? | Didn't you say you were going to get a new phone? |
| 아 맞다! 짜잔~ | Oh, that's right! Check it out~ |
| 전에 쓰던 건? 고장난 거야? | What about your old one? Is it broken? |
| 몇 번 수리를 맡겨봤는데 | I've tried getting this phone fixed a couple of times |
| 무슨 이유에서인지 계속 문제가 생겨 | But for some reason, it kept causing problems |
| 새거를 살 때가 됐다 싶었지 | So I decided it was time to get a new one |

**핵심 표현**

* **didn't you say ~?** — ~라고 하지 않았어?

    * _Didn't you say you were moving? / going to break up with her?_

* **change (×) → get a new one (○)** — 새로 산다는 표현은 `get` 사용
* **check it out** — 짜잔~ / 이것봐라~ / 알아봐
* **get + something + p.p.** — 사역 (맡기다/받다)

    * _I got this phone fixed._ (수리 받았어)
    * _I got my clothes dry-cleaned._ (드라이클리닝 맡겼어)

* **try to V vs try + ing**

    * `try to V` — ~하려고 노력하다
    * `try + ing` — ~을 시도해보다

* **keep + ing** — 계속/자꾸 반복해서 일어나다
* **act up** — 오작동을 일으키다

## 대화가이드 #03 — 전화 받을 때

**상황:** 대화 중 전화가 왔을 때 예의 있게 처리하는 상황

| 한국어 | 영어 |
| --- | --- |
| 안 받을 거야? 어서 받아 | Aren't you going to get that? Go ahead, get it. |
| 잠시만요, 통화 좀 하고 올게 | Could you hold on for a second? I need to take this call. |
| 어서 해, 천천히 해 | Go ahead. Take your time. |
| 삼촌이랑 친해? | Are you close with your uncle? |
| 가끔 전화를 하시더라고 | He's been calling me from time to time. |

**핵심 표현**

* **go ahead** — 어서 자유롭게/주저하지 말고 해
* **hold on** — 기다려줄래? _(hold on for a minute / a second)_

    * `hold` (without on) → 더 격식체 _(Could you hold please?)_

* **I need to take this call** — 전화가 와서 받아야 할 때

    * 문자 보고 전화할 때는: _make a phone call_

* **be close with someone** — ~와 친하다
* **from time to time / every now and then** — 가끔

    * _sometimes_보다 완료시제와 더 잘 어울림

* **전화 관련 표현**

    * 전화 걸다: _call someone / dial a number_
    * 전화가 오다: _get a call from ~_
    * _I got a call from a friend._ / _A friend called me._

## 대화가이드 #04 — 처음 보는 강한 인상 표현

### Part 1 — 처음 표현

**상황:** 믿을 수 없이 고집 센 사람을 봤을 때

| 한국어 | 영어 |
| --- | --- |
| 나 저렇게 고집 센 사람 처음 봐 | I've never seen such a stubborn person. / I've never seen anyone so stubborn. |
| 나도 저런 사람 처음 보는 것 같아 | I've never seen anyone like that. / I don't think I've ever seen anyone like that. |
| 누구랑 얘기만 하는데도 이렇게 힘든 건 처음이야 | I've never had such a hard time just talking to someone. |

**핵심 표현**

* **I've never ~** — ~처음이야 / ~한 적 없어
* **I don't think I've ever ~** — 처음인 것 같아 _(조심스럽고 진지한 표현)_
* **have a hard time + ing** — ~하는 게 힘든 시간이다
* **be stressed / be under stress**

    * _I'm very stressed these days._ / _I'm under a lot of stress these days._

### Part 2 — 최고/최악 표현

| 한국어 | 영어 |
| --- | --- |
| 나 이렇게 맛있는 파스타는 처음이야 | I've never had pasta this good. / This is the best pasta I've ever had. |
| 저렇게 많이 먹는 사람 처음 봐 | I've never seen anyone eat so much. |
| 나 너처럼 많이 먹는 사람 처음 봐 | I've never seen anyone eat as much as you. |

**강조 표현**

```
This is the best ~         (기본)
This has to be the best ~  (확신)
This must be the best ~    (강한 확신)
This has got to be the best ~ (가장 강한 강조)
```

* **in years** — 몇 년 만에 _(That was the best movie I've seen in years.)_

## 대화가이드 #09 — 음식 사양하기

**상황:** 해물을 먹지 않는데 모르고 시킨 상황

| 한국어 | 영어 |
| --- | --- |
| 이거 좀 먹어봐요, 맛나요 | Hey, try this, it's really good. |
| 아뇨, 전 됐어요. 해물을 잘 안 먹거든요 | No, I'm good. Actually, I don't really eat seafood. |
| 왜 아무 말도 안 했어요? 다른 거 시켰을 텐데 | Really? Why didn't you say anything? We would have gotten something else. |
| 아니에요, 다른 거 잘 먹고 있어요 | No, it's really OK. I'm really enjoying everything else. |
| 그래도 말했었어야지. 시킬 거 많았는데 | Still, you should have told us. There were tons of other things we could have ordered. |
| 저는 다들 그냥 원하는 거 먹었으면 해요 | I'd really prefer it if everyone just got what they want. |

**핵심 표현**

* **I'm good** — 사양할 때 가장 자연스러운 표현
* **would have gotten** — ~했을 텐데 _(if 생략된 가정법)_
* **tons of = a lot of**
* **could have ordered** — 시킬 수 있었을 텐데 _(가능성)_
* **I'd really prefer it if ~** — ~했으면 정말 좋겠어
* **I really don't mind / It's really not a problem**

## 대화가이드 #10 — 예약 변경 요청

**상황:** 병원 예약을 이번 주 안에 당기려는 상황

| 한국어 | 영어 |
| --- | --- |
| 김 박사님이랑 상담 예약을 하려고요 | I need to make an appointment with Dr. Kim. |
| 다음 주 월요일에 가능합니다 | He's available next Monday. |
| 이번 주 내로 만날 수 있는 방법은 없을까요? | Is there any way I can see him this week? |
| 예약 취소가 있지 않는 한, 월요일이 제일 빨라요 | As long as there are no cancellations before then, Monday is the best we can do. |
| 무리한 부탁인 건 아는데, 다른 분 예약을 변경할 수는 없을까요? | I know it's a lot to ask, but can't you reschedule one of the others? |
| 제가 할 수 있는 건 해보겠는데요, 장담은 못 드려요 | I'll see what I can do, but I can't make any promises. |

**핵심 표현**

* **be available** — 이용 가능한, 시간 여유가 있는
* **as long as** — ~하는 한
* **Is there any way I can ~?** — ~할 수 있는 방법이 있을까요?
* **a lot to ask** — 무리한 부탁
* **I'll see what I can do** — 할 수 있는 건 해볼게요
* **I can't make any promises / I can't promise anything**
* **get into a lot of trouble** — 혼나다, 곤란해지다

## 대화가이드 #11 — 공포영화를 왜 좋아해?

**상황:** 공포영화를 무서워하면서 계속 보는 이유를 설명하는 대화

| 한국어 | 영어 |
| --- | --- |
| 난 네가 공포영화를 왜 그렇게 좋아하는지 모르겠어 | I don't understand why you like horror movies so much. |
| 무섭다면서 왜 계속 봐? | You say they scare you, then why do you keep watching them? |
| 그 스릴을 즐기는 거지. 롤러코스터도 마찬가지야 | I enjoy that rush / thrill. It's the same with roller coasters. |
| 아, 그렇게는 생각 안 해봤는데. 그것도 말 되네 | Oh, I haven't thought of it that way. But that makes sense. |

**핵심 표현**

* **I haven't thought of it like that (that way)**
* **I've never looked at it that way**
* **that makes sense** — 말이 되네, 이해돼

## 대화가이드 #16 — 쿠폰 / 할인 안내

**상황:** 쿠폰 사용 방법을 안내하고 확인하는 대화

| 한국어 | 영어 |
| --- | --- |
| 웹사이트에서 쿠폰을 다운로드해야 할인을 받을 수 있어요 | You can only get the discount if you have the coupon downloaded through our website. |
| 회원으로 가입을 해야 할 수 있어요. 이해가 되셨으면 좋겠네요 | You need to be signed up as a member to do that. I hope that makes sense. |
| 제가 맞게 이해한 거면, 웹사이트 가서 쿠폰 받으면 되는 거죠? | So if I understood you correctly, all I need to do is go to your website and download the coupon? |

**핵심 표현**

* **I hope that makes sense** — 무언가 복잡하게 설명한 후 이해 확인 _(Do you understand?보다 자연스럽고 압박 없음)_
* **all I need to do is ~** — ~하기만 하면 된다
* **if I understood you correctly** — 제가 맞게 이해한 거라면
* **but to do that, you first need to ~** — 근데 그러려면, 먼저 ~해야 돼

## 대화가이드 #17 — 낯이 익은 사람

**상황:** 길에서 낯익은 사람을 보고 대화하는 상황

| 한국어 | 영어 |
| --- | --- |
| 뭘 그렇게 계속 쳐다봐? | What are you keep looking at? |
| 저 사람 낯이 좀 익어서, 아는 사람 같아 | He looks kind of familiar. I think I know him. |
| 저기 버스 정류장에 앉아있는 저 사람? | The guy over there, sitting at the bus stop? |
| 아니, 그 옆에, 안경 쓰고 자기 핸드폰 보고 있는 사람 | No, the guy next to him, with glasses and looking at his phone. |
| 가서 말 걸어봐 | Why don't you go talk to him? |
| 아니야, 됐어 | Nah, it's OK. / Forget it. |

**핵심 표현**

* **look kind of familiar** — 낯이 익다
* **관계절 생략** — 묘사할 때 흔히 생략 가능

    * _I saw a man **who was** smoking → I saw a man smoking_

* **거의 다 / 대부분**

    * _almost all (people / my friends)_ — 거의 다
    * _most (of my friends)_ — 대부분
