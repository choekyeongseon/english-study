## 2026-03-12 세션 정리 (메인 페이지)

### 📌 주제

* 오늘 하루 설명하기 (서버 장애 사건)
* 직업 소개 및 업무 설명
* 자기소개 (거주지, 가족, 취미, 영어 공부 이유)

### 🔑 핵심 표현 모음

#### 1. 오늘 하루 / 서버 장애

| 한국어 | 영어 |
|--------|------|
| 서버 장애 | server outage / server failure |
| 복구하다 | fix it / restore it / get it back up |
| 출근하자마자 | as soon as I got to work |
| 하루 종일 고치는데 썼다 | spent the whole day fixing it |
| 가볍게 점심을 먹었다 | grabbed a light lunch |
| 지금은 복구됐다 | everything is back up and running now |
| 정신없이 바쁜 | hectic / chaotic / crazy busy |

**✅ 완성 문장:**

> "As soon as I arrived at the office this morning, I discovered the server issue, and it took all day to resolve. Luckily, everything is back up and running now."

#### 2. 직업 소개

| 한국어 | 영어 |
|--------|------|
| ~로 일하다 | work as ~ |
| 내 주요 업무는 ~이다 | my main responsibility is to ~ |
| ~팀을 담당하다 | be in charge of the ~ team |
| 문제가 생기면 내가 고친다 | when something breaks, I'm the one who fixes it |
| IT 비전공자 | non-technical people / people outside of IT |

**✅ 완성 문장:**

> "I've been working as a software engineer for a cloud company, and my main responsibility is ensuring the software our developers build runs properly for users. When something breaks, I'm the one who fixes it."

#### 3. 자기소개

| 주제 | 완성 문장 |
|------|----------|
| 거주지 | "My wife and I live in the Dongtan area, just near the newly built Hwaseong Dongtan Library." |
| 학원 위치 | "The English academy is really close to my place — just a 10 minute walk!" |
| 가족 | "We don't have any kids yet, but we're hoping to start a family soon!" |
| 취미 | "I'm really into weight training — I try to hit the gym about 3 times a week." |
| 영어 공부 이유 | "I started learning English for my career — I'd really like to work in the U.S. someday." |

### 📚 문법 포인트

1. `spend + 동명사(-ing)`: ✅ `I spent the whole day fixing it.`
2. 간접의문문 어순: ✅ `She asked how my day was.`
3. `work as`: ✅ `I work as a software engineer.`
4. `this morning` vs `today morning`: ✅ `this morning` (고정 표현)
5. 주어-동사 수일치: ✅ `The software our developers build **runs** properly.`

### ⚠️ 자주 틀리는 스펠링

| 틀린 표현 | 올바른 표현 |
|----------|-----------|
| quit | quite |
| hole | whole |
| conputer | computer |
| tranning | training |
| jet | yet |

### 💡 유용한 표현 - 상황별

#### 부상 설명

| 한국어 | 영어 |
|--------|------|
| 계단을 내려가다가 넘어졌다 | she fell while walking down the stairs |
| 팔이 부러졌다 | she broke her arm |
| 수술을 받아야 했다 | she had to have surgery on her arm |
| ~를 돌보다 | take care of ~ / look after ~ |
| 그래서 수업에 못 갔어 | so I couldn't make it to class |

#### 이동 시간

| 한국어 | 영어 |
|--------|------|
| 걸어서 10분 | a 10 minute walk / 10 minutes on foot |
| 차로 10분 | a 10 minute drive / 10 minutes by car |
| 걸어서 갈 수 있는 거리 | within walking distance |
